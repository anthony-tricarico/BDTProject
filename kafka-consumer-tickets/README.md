# Tickets API

This API exposes the following endpoints on port 8001 of local host:

1. `/latest_tickets` -> returns the latest measurement
2. `/stream_tickets` -> returns the latest 100 measurements

## Examples

`/latest_tickets` and `/stream_tickets` are easily accessible by sending requests to `http://localhost:8001/latest_tickets` and `http://localhost:8001/stream_tickets`, respectively.
